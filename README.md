## 🌍 The Problem
Manual waste tracking in Gombe is slow and lacks real-time data. This leads to environmental hazards and inefficient cleanup schedules.

The Solution
**SmartWaste AI** is a full-stack web application that digitizes the reporting process. Using a PHP backend and Python-based AI logic, it allows citizens to report waste via photo uploads and automatically categorizes the data for government action.

## 🚀 Key Features
* **AI Waste Classification:** Automatically identifies waste types (Plastic, Organic, etc.).
* **Admin Dashboard:** Live statistics and analytics for environmental officers.
* **Data Export:** Generate CSV/Excel reports for official planning.
* **Mobile-Friendly UI:** Clean, modern interface designed for easy reporting.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP (XAMPP Environment)
* **Database:** MySQL / MariaDB
* **AI Engine:** Python (Image Processing Logic)
